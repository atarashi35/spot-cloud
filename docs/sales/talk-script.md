# 訪問トークスクリプト（文化拠点全般向け）

対象: 書店・ミニシアター・ライブハウス・劇場・ギャラリー・神社・市民楽団・劇団・茶室・バーなど、「続いてほしい」と思われている街の文化拠点・活動全般。個人の小規模書店だけでは母数が少ないため、最初から文化拠点全般を対象とする。

## 大原則

- **メールではなく現地に行く。その場所のものを一つ体験・購入してから話す。**（本を買う／一杯飲む／公演を観る／一品頼む）
- 売り込まない。「活動を続けるための仕組みを作っていて、最初の10拠点を探している」と正直に言う。
- その場で契約を迫らない。目的は「次のアポ」または「その場でのデモ登録」。
- **断られたら必ず理由を聞いて記録する。** 断られた記録こそが今月の最重要データ。

## 30秒ピッチ

> 「ファンや常連さんが月300円から、応援会員になれる仕組みを作っています。
> このPOP（見せる）を目立つ場所に置くだけで、QRから入会して、
> 毎月の会費の約87%がそのまま振り込まれます。
> 初期費用も月額もゼロ。埼玉の移動式茶室プロジェクトでも使われ始めています。
> 最初の10拠点を探していて、◯◯さんのところに一番に置いてほしいんです。」

## 訪問の流れ（15分）

1. その場所のものを体験・購入する／好きなところを一つ具体的に言う（2分）
2. 30秒ピッチ + 紙の提案資料を渡す（1分）
3. スマホでデモ: ページのサンプル → 入会フロー → 会員証 → POP印刷画面（5分）
4. 質問に答える（5分）
5. クロージング: 「今ここで5分で登録できます。まず非公開で作ってみませんか」（2分）
   - 渋ったら「来週、登録した状態の画面を持ってまた来ます」で次アポ。

## よくある反応と返し

| 反応 | 返し |
|---|---|
| 「手数料10%は高い」 | 「固定費ゼロなので、会員が0人なら費用も0円です。比較対象は手数料ではなく、何もしなければ入らなかった月◯円です」 |
| 「ファン・常連にお金の話をしにくい」 | 「だからPOPなんです。あなたが口で頼む必要はなく、置いておくだけ。入りたい人だけが入ります」 |
| 「返礼・特典を考えるのが大変」 | 「特典は不要です。会員証と『支えている関係』そのものが価値。月300円は、返礼を気にせず受け取れる金額として設定しています」 |
| 「ITが苦手」 | 「登録は私がこの場で代行します。日々の作業はゼロ。投稿したくなったら写真1枚で十分です」 |
| 「既に物販/ファンクラブ/クラファンをやってる」 | 「それは商品や一度きりの支援で、これは『続いてほしい』への毎月の会費です。併存できます。むしろ既存のファンが移行しやすいです」 |
| 「うちはまだ大丈夫」 | 深追いしない。「応援会員が必要になる前に始めるほど効きます。半年後にまた来ます」と言って撤退。記録だけ残す |

## 断られ理由の分類（CSVに必ず記録）

1. **手数料** — 価格設計の問題
2. **ファン・常連に頼みにくい** — ポジショニング/POPの問題
3. **IT不安** — オンボーディングの問題
4. **既存手段あり** — 差別化の問題
5. **興味なし** — ICP選定の問題

→ 30拠点訪問して同じ分類が10回出たら、それが直すべき箇所。同時に、**どの業態(書店/音楽/演劇/神社/個人活動…)で契約率が高いか**を記録し、伸びる業態に寄せる。

## Day 30 ゲート（事前登録・変更不可）

- 訪問30拠点 / 契約10拠点 / 稼働(公開+POP設置)5拠点
- 契約5拠点未満 → ポジショニングか価格を見直す（業態を絞るのではなく、最も契約率の高い業態に注力する）
- どの業態の契約率が高かったかを集計し、第2ラウンドはその業態を重点的に回る（コード変更なし）
