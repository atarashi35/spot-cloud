# 訪問トークスクリプト（独立書店向け）

## 大原則

- **メールではなく店に行く。本を1冊買ってから話す。**
- 売り込まない。「店を続けるための仕組みを作っていて、最初の10店を探している」と正直に言う。
- その場で契約を迫らない。目的は「次のアポ」または「その場でのデモ登録」。
- **断られたら必ず理由を聞いて記録する。** 断られた記録こそが今月の最重要データ。

## 30秒ピッチ

> 「常連さんが月300円から、お店の応援会員になれる仕組みを作っています。
> レジ横にこのPOP（見せる）を置くだけで、お客さんがQRから入会して、
> 毎月の会費の約87%がお店に振り込まれます。
> 初期費用も月額もゼロで、埼玉の歴史ある神社でも使われ始めています。
> 最初の10店を探していて、◯◯さんの店に一番に置いてほしいんです。」

## 訪問の流れ（15分）

1. 本を買う・店の好きなところを一つ具体的に言う（2分）
2. 30秒ピッチ + 紙の提案資料を渡す（1分）
3. スマホでデモ: 店ページのサンプル → 入会フロー → 会員証 → POP印刷画面（5分）
4. 質問に答える（5分）
5. クロージング: 「今ここで5分で登録できます。まず非公開で作ってみませんか」（2分）
   - 渋ったら「来週、登録した状態の画面を持ってまた来ます」で次アポ。

## よくある反応と返し

| 反応 | 返し |
|---|---|
| 「手数料10%は高い」 | 「固定費ゼロなので、会員が0人なら費用も0円です。比較対象は手数料ではなく、何もしなければ入らなかった月◯円です」 |
| 「常連にお金の話をしにくい」 | 「だからPOPなんです。店主さんが口で頼む必要はなく、置いておくだけ。入りたい人だけが入ります」 |
| 「ITが苦手」 | 「登録は私がこの場で代行します。日々の作業はゼロ。投稿したくなったら写真1枚で十分です」 |
| 「既に選書サブスク/ファンクラブをやってる」 | 「それは商品の販売で、これは『店を続けてほしい』への会費です。併存できます。むしろ既存のお客さんが移行しやすいです」 |
| 「うちはまだ大丈夫」 | 深追いしない。「応援会員が必要になる前に始めるほど効きます。半年後にまた来ます」と言って撤退。記録だけ残す |

## 断られ理由の分類（CSVに必ず記録）

1. **手数料** — 価格設計の問題
2. **客に頼みにくい** — ポジショニング/POPの問題
3. **IT不安** — オンボーディングの問題
4. **既存手段あり** — 差別化の問題
5. **興味なし** — ICP選定の問題

→ 30店訪問して同じ分類が10回出たら、それが直すべき箇所。5(興味なし)が最多なら書店仮説の棄却を検討（Day30ゲート）。
