# 営業キット

書店向け「応援会員」手売り営業の道具一式。**最初の成果物はコードではなく、埋まった30店リスト。**

| ファイル | 用途 |
|---|---|
| [proposal-a4.html](proposal-a4.html) | 紙1枚の提案資料。ブラウザで開いて A4・余白なしで印刷。フッターの連絡先を自分の連絡先に書き換えてから使う |
| [store-list.csv](store-list.csv) | 30店リスト 兼 商談記録 兼 KPI記録。Googleスプレッドシートにインポートして使う |
| [talk-script.md](talk-script.md) | 30秒ピッチ・訪問の流れ・返し・断られ理由の分類・Day30ゲート |

## 週次でやること

1. 訪問8店以上（店リストの「訪問日・反応・結果・断られ理由」を当日中に記入）
2. 稼働店の会員数をCSVに記録（毎週金曜）
3. 週末は稼働店の店頭に立ち、入会の瞬間／スルーの瞬間を観察してメモ

## 検証仮説（この順に重要）

- **H1**: 店主は店頭で応援会員を勧めるか（POP設置率・口頭勧誘の有無）
- **H2**: 常連はモノなしで月500円払うか（告知到達から60日で30人）
- **H3**: 3ヶ月残存90%以上か
- **H4**: 会員証はSNSにシェアされるか（加入者の10%）
