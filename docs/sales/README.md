# 営業キット

文化拠点全般向け「応援会員」手売り営業の道具一式。対象は書店・ミニシアター・ライブハウス・劇場・ギャラリー・神社・市民楽団・劇団・茶室・バーなど。個人の小規模書店だけでは母数が少ないため、最初から文化拠点全般を対象とする。**最初の成果物はコードではなく、埋まった30拠点リスト。**

| ファイル | 用途 |
|---|---|
| [proposal-a4.html](proposal-a4.html) | 紙1枚の提案資料。ブラウザで開いて A4・余白なしで印刷。フッターの連絡先を自分の連絡先に書き換えてから使う |
| [store-list.csv](store-list.csv) | 30拠点リスト 兼 商談記録 兼 KPI記録。Googleスプレッドシートにインポートして使う |
| [talk-script.md](talk-script.md) | 30秒ピッチ・訪問の流れ・返し・断られ理由の分類・Day30ゲート |

## 週次でやること

1. 訪問8拠点以上（リストの「訪問日・反応・結果・断られ理由」を当日中に記入）
2. 稼働拠点の会員数をCSVに記録（毎週金曜）
3. 週末は稼働拠点の現地に立ち、入会の瞬間／スルーの瞬間を観察してメモ

## 検証仮説（この順に重要）

- **H1**: 運営者は現地で応援会員を勧めるか（POP設置率・口頭勧誘の有無）
- **H2**: 常連はモノなしで月300円払うか（告知到達から60日で30人）
- **H3**: 3ヶ月残存90%以上か
- **H4**: 会員証はSNSにシェアされるか（加入者の10%）
